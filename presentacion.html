    Stack Tecnológico

    Frontend
    • Vue.js y JavaScript: Interfaz de usuario dinámica, reactiva y componentizada
    • Tailwind CSS: Framework utility-first para desarrollo rápido y consistente
    Backend
    • Laravel (PHP): Framework robusto con estructura MVC, ORM (Eloquent)
    • PostgreSQL: Base de datos relacional escalable y fiable
    Interconexión
    • Inertia.js: "Pegamento" para SPAs con routing de Laravel y Vue.js
    Tecnologías Avanzadas
    • Progressive Web App: silviolleite/laravelpwa
    • IA - Reconocimiento de Voz: huggingface/transformers con Whisper Tiny
    • IA - Generación de Presupuestos: API de OpenAI via openai-php/laravel

    Diseño y Organización del Código

    Arquitectura General
    • SPA monolítica estructurada, combinando Laravel, Inertia.js y Vue.js
    Backend (Laravel)
    • Modelos: Client, Budget, User, Subscription con Eloquent ORM
    • Controladores: BudgetController, ClientController, PromptController
    • Políticas de Autorización: ClientPolicy, BudgetPolicy
    • Patrón Repositorio: En SubscriptionController
    Frontend (Vue.js)
    • Components/: Elementos reutilizables (Banner.vue, creditsTile.vue)
    • Pages/: Vistas completas correspondientes a rutas
    • Layouts/: Estructuras generales (AppLayout.vue)
    • Servicios: Notify.js, speechRecognition.js

    Librerías

    Integradas en Framework
    • Laravel: Eloquent ORM, Validator, rutas, middleware
    • Inertia.js: useForm para gestión de formularios
    • Tailwind CSS: Clases de utilidad para interfaces
    Específicas del Proyecto
    • silviolleite/laravelpwa: Funcionalidad PWA
    • huggingface/transformers: Reconocimiento de voz
    • openai-php/laravel: Cliente para API de OpenAI
    • dayjs: Manipulación de fechas
    • Heroicons: Iconografía SVG

    Funcionalidades Clave Implementadas

    Frontend
    • Generación Asistida por IA: Interfaz interactiva con indicador de créditos
    • Reconocimiento de Voz: Procesamiento local con privacidad y optimización
    • Gestión de Suscripciones: Visualización de créditos y planes
    • PWA y Notificaciones: Banner.vue, Service Worker, manifest.json
    • UX: Tailwind CSS, modo claro/oscuro, responsive, accesibilidad
    Backend
    • Gestión de Clientes: Relaciones complejas, Soft Deletes, reasignación
    • IA para Presupuestos: Patrón Repositorio, validación, pipeline de generación
    • Autorización: Políticas granulares, validación robusta con reglas complejas

    Despliegue y Puesta en Producción

    Infraestructura
    • VPS (Droplet) en Digital Ocean
    • Docker y Docker Compose con imagen PHP 8.2-Apache
    • Apache configurado con HTTPS y redirección HTTP
    CI/CD
    • Travis CI
    • Automatizar pruebas y despliegue
    • Script deploy.sh con rsync y gestión de permisos
    Seguridad
    • HTTPS: Certificado SSL Sectigo
    • ModSecurity: WAF con OWASP Core Rule Set
    • Fail2Ban: Baneo automático de IPs maliciosas
    • Cloudflare: Protección perimetral adicional

    Testing

    Estrategia
    • Enfoque: Pruebas End-to-End (E2E)
    • Herramienta: Cypress
    • Automatización: Integrado en CI/CD (Travis CI)
    Suite de Pruebas
    • 01_home.cy.js: Carga de aplicación
    • 02_register.cy.js: Registro de usuario
    • 03_login.cy.js: Inicio de sesión
    • 04_costs.cy.js: Creación/edición de costes
    • 05_clients.cy.js: Gestión de clientes
    • 06_budgets.cy.js: Gestión de presupuestos
    • 99_deleteUser.cy.js: Limpieza de datos
