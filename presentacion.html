Budget App es una aplicación web para la gestión de presupuestos

Idea

Quién

  • Pequeños Empresarios y Autónomos
  • Emprendedores
  • Profesionales independientes

Problema:

  • Los profesionales españoles dedican hasta 10,5% del tiempo laboral a tareas administrativas
    • (Sage, n.d.)
  • Esta cifra escandalosa equivale a 4 horas semanales o 192 horas anuales completamente improductivas
  • Un autónomo con tarifa media de 35€/hora está perdiendo potencialmente 6.720€ anuales en tareas que no generan valor directo
  • Una encuesta a freelancers de Estados Unidos (Independent Economic Council) indicó que más del 30% de trabajadores independientes utiliza Microsoft Word o programas similares para crear sus facturas. Otro 20% recurre a plantillas prediseñadas
  • Esta improvisación genera inconsistencias en el formato, errores en cálculos y una imagen poco profesional ante los clientes potenciales

Cliente y Mercado Objetivo

Segmento de Clientes

  • Autónomos y profesionales independientes
  • Microempresas (0-5 empleados)
  • Emprendedores en fase inicial
  • Pequeños negocios, especialmente de servicios
  • 2-4 M España

Perfil Cliente

  • Edad: 25-50 años
  • Digitalmente activos (uso diario de herramientas)
  • Gestión de 5-50 presupuestos/mes
  • Sensibles al precio, pero valoran su tiempo
  • Buscan profesionalizar su imagen comercial

Producto

¿Qué hace Budget App?

  • Gestión integral de presupuestos
  • Creación de presupuestos mediante interfaz intuitiva, asistente IA o reconocimiento de voz
  • Exportación a PDF
  • Simplicidad de uso
  • Acceso desde cualquier dispositivo

Propuesta de Valor

  • Ahorro de tiempo significativo
  • Presupuestos con aspecto profesional
  • Precisión garantizada en cálculos y totales
  • Experiencia de usuario intuitiva sin curva de aprendizaje
  • Creación automática con la integración de IA
  • Solución multiplataforma (móvil, tablet, escritorio)
  • Trazabilidad completa del ciclo de presupuestos

Diseño y Organización del Código

Arquitectura General
  • SPA monolítica estructurada, combinando Laravel, Inertia.js y Vue.js
  • Estructura MVC
Frontend
  • Vue.js y JavaScript: Interfaz de usuario dinámica, reactiva y componentizada
  • Tailwind CSS: Framework utility-first para desarrollo rápido y consistente
Backend
  • Laravel (PHP): Framework robusto y muy usado con estructura MVC, ORM (Eloquent)
  • PostgreSQL: Base de datos relacional escalable y fiable. Servicio Externo

Librerías

Integradas en Framework
  • Laravel: Eloquent ORM, Validator, Sanctum
  • Inertia.js: Back + Front
  • Tailwind CSS: Facilidad de uso
Específicas del Proyecto
  • silviolleite/laravelpwa: Funcionalidad PWA
  • huggingface/transformers: Reconocimiento de voz
  • openai-php/laravel: Cliente para API de OpenAI
  • dayjs: Manipulación de fechas
  • paparse y pdfjs-dist: Parsear costes
  • Heroicons: Iconografía SVG

Testing

Estrategia
  • Enfoque: Pruebas End-to-End (E2E)
  • Herramienta: Cypress
  • Automatización: Integrado en CI/CD (Travis CI)
Suite de Pruebas
  • 01_home.cy.js: Carga de aplicación
  • 02_register.cy.js: Registro de usuario
  • 03_login.cy.js: Inicio de sesión
  • 04_costs.cy.js: Creación/edición de costes
  • 05_clients.cy.js: Gestión de clientes
  • 06_budgets.cy.js: Gestión de presupuestos
  • 99_deleteUser.cy.js: Limpieza de datos

Funcionalidades Clave Implementadas

Frontend

Despliegue y Puesta en Producción

Infraestructura
  • VPS (Droplet) en Digital Ocean
  • Docker y Docker Compose con imagen PHP 8.2-Apache
  • Apache configurado con HTTPS y redirección HTTP
  • Dominio .software con ssl
CI/CD
  • Travis CI
  • Automatizar pruebas y despliegue
  • Script deploy.sh con rsync y gestión de permisos
Seguridad
  • HTTPS: Certificado SSL Sectigo
  • ModSecurity: WAF con OWASP Core Rule Set
  • Fail2Ban: Baneo automático de IPs maliciosas
  • Cloudflare WAF: Protección perimetral adicional
